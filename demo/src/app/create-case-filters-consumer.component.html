Create Case Filters component

This component is used to determine the jurisdiction, case type and event when creating a case. The elements of the contract are:

  • Input fields:
    • jurisdictions - a collection of jurisdictions - including nested casetypes and events

      Jurisdiction model

    • isDisabled - boolean which controls the `disable` state of the button
    • startButtonText - label of the button
  • Output fields:
    • selectionSubmitted - a notification event about a CreateCaseFiltersSelection being submitted containing an event with:
      • jurisdictionId: string
      • caseTypeId: string
      • eventId: string
    • selectionChanged - a notification event about selection change on any of the dropdowns
