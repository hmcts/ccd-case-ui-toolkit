Workbasket Filters component

This component is used to set the jurisdiction, case type, state and keyword when searching cases. The elements of the contract are:

  • Input fields:
    • jurisdictions - a collection of jurisdictions - including nested casetypes and events

      Jurisdiction model

    • defaults - a jurisdiction
  • Output fields:
    • onApply - a notification event about search filters being submitted containing an event with:
      • selected: Jurisdiction
      • queryParams: QueryParams
    • onReset - a notification event about resetting search filters
