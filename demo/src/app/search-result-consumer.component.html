Search Result component

This component is used to list cases following a search action. The elements of the contract are:

  • Input fields:
    • jurisdiction

      Jurisdiction model

    • caseType

      CaseType model

    • caseState

      CaseState model

    • caseFilterFG

      FormGroup

    • resultView

      SearchResultView model

    • paginationMetadata

      PaginationMetadata model

    • metadataFields - a collection of strings
  • Output fields:
    • changePage - a notification event about paging navigation containing:
      • selected:
        • init?: boolean
        • jurisdiction?: Jurisdiction
        • caseType?: CaseType
        • caseState?: CaseState
        • formGroup?: FormGroup
        • metadataFields?: string[]
        • page?: number
      • queryParams:
        • case-state: string
        • case-type: string
        • jurisdiction: string
