Case List Filters component

Wrapper component for ccd-workbasket-filters. Service call to get "jurisdictions" is done here.

This component is used to set the jurisdiction, case type, state and keyword when searching cases. The elements of the contract are:

  • Input fields:
    • defaults - an object consisiting of jurisdiction, casetype and state id's 
      {{json.stringify(defaults, null, '\t')}}
  • Output fields:
    • onApply - a notification event about search filters being submitted containing an event with:
      • selected: Jurisdiction
      • queryParams: Object
    • onReset - a notification event about search filters being reset
